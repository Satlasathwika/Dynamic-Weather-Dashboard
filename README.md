Dynamic Weather Dashboard 🌦️

The Dynamic Weather Dashboard is a responsive and interactive web application that provides real-time weather updates for any city worldwide. Built using HTML, CSS, and JavaScript, the project integrates a weather API to fetch live weather data and display it in a clean, user-friendly interface.
Users can search for weather conditions by city name or use geolocation to automatically detect their current location. The dashboard dynamically updates weather information such as temperature, humidity, wind speed, and overall weather conditions with visually appealing icons and backgrounds.

🚀 Features

🌍 Search weather by city name
📍 Geolocation support for current location weather
🌡️ Real-time temperature and weather updates
💨 Displays humidity, wind speed, and conditions
🎨 Dynamic UI with weather-based visuals
📱 Fully responsive design for mobile and desktop
🕘 Recent search history support
🛠️ Tech Stack
HTML5 – Structure of the application
CSS3 – Styling and responsive layout
JavaScript (ES6+) – API integration and interactivity
Weather API – Fetching live weather data
📌 Purpose
This project demonstrates how to build a real-world front-end application using API integration, responsive design principles, and dynamic DOM manipulation while maintaining a smooth user experience.
